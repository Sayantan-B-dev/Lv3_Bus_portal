Full Docs ↗
GET/api/routes
Fetch all active bus routes. Supports pagination, filtering by type, and search by source/destination.
?q=kashmere?type=AC?page=1
GET/api/routes/{id}
Get a single route with all stop sequences, fare slabs, and schedule metadata by route ID.
{id}: INTincludes=stops,fares
GET/api/search
Autocomplete search across routes and stops. Returns ranked results by relevance score.
?q=dwarka?limit=10
POST/api/admin/routes
Create a new bus route. Requires admin JWT authentication via Google OAuth 2.0 bearer token.
Authorization: BearerContent-Type: JSON
